# Daily Web Practice

This is a very easy project for beginners.

You can use this repository to:
- Practice small daily edits
- Write simple notes
- Learn how a basic web project works
- Build a habit of GitHub commits

How to use:
1. Open any file
2. Make a small change
3. Save the file
4. Commit your update

Goal:
Practice something small every day.

Notes:
Write your daily practice notes below.
